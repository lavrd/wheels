# wheels
Change wheels of car like a boss

#### usage

##### pre start
1. start mongodb

##### npm
1. `cd ./server && npm i && npm run dev`
2. `cd ./dashboard && npm i && npm start`
3. open `http://localhost:1101` in your browser

##### docker-compose
1. `docker-compose up -d`
2. open `http://localhost:1101` in your browser
