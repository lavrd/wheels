# wheels
Change wheels of car like a boss

#### usage

##### npm
1. start `mongodb`
2. `cd ./server && npm i && npm run dev`
3. `cd ./dashboard && npm i && npm start`
4. open `http://localhost:1101` in your browser

##### docker-compose
1. `docker-compose up -d`
2. open `http://localhost:1101` in your browser
